## Columns with generated values-------------------

بنابر نیاز پروژه ممکنه لازم بشه که مقدار یک ستون غیر کلید بصورت خودکار (نه دستی) ساخته بشه برای این منظور بالای property مورد نظر از دکوراتور `@Generated()` استفاده می کنیم:

```ts
@Entity()
export class User {
    @PrimaryColumn()
    id: number

    @Column()
    @Generated("uuid")
    uuid: string
}
```

در مثال بالا از `uuid` استفاده شده است ، علاوه بر uuid می توان از "`increment`", "`identity`" (Postgres 10+ only) and "`rowid`" (CockroachDB only) نیز استفاده کرد. اگر چه در بعضی از دیتابیس ها محدودیت هایی برای اینکار وجود دارد به عنوان مثال برخی دیتابیس ها تنها اجازه وجود یک ستون خود مقدار دهی شونده رو می دهند و برخی دیگر تنها به ستونی که کلید هست اجازه افزایش خودکار رو می دهند.
